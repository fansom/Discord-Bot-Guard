# AI Governance

This project uses local AI collaboration scaffolding under .dev-flow/ and .ai-harness/tasks/.

- .dev-flow/spec.md defines the current workflow goal.
- .dev-flow/tasks/ stores implementation task files and context packs.
- .dev-flow/handoff/ stores agent handoff templates.
- .ai-harness/tasks/ stores local task records when using the downstream Harness process.

Runtime code changes should come from a concrete task design and should be reviewed before acceptance.
